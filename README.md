# CYBERTAG Showcase

Отдельный многостраничный сайт CYBERTAG в чёрно-красном стиле.

## Реализовано

- страницы оборудования, комплектов, софта, акций и контактов;
- отдельная вкладка корзины со счётчиком, удалением и очисткой;
- отдельная страница входа перед оформлением заказа;
- автоматический перенос состава заказа в форму контактов;
- адаптивная версия для компьютеров и телефонов;
- кнопки загрузки на странице софта без действия;
- хранение демонстрационной корзины и входа в `localStorage`.
