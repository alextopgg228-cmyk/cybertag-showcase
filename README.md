# CYBERTAG Showcase

Многостраничный сайт CYBERTAG с серверной регистрацией и входом через Microsoft SQL Server 2022 Express. Исходное чёрно-красное оформление сохранено.

## Реализовано

- каталог оборудования, комплекты, софт, акции и контакты;
- отдельная корзина со счётчиком, удалением и очисткой;
- регистрация с сохранением пользователя в таблицах `Customers` и `Users`;
- bcrypt-хэширование паролей;
- вход по логину или email;
- серверные сессии в таблице `Sessions` и защищённая `HttpOnly` cookie;
- выход с удалением сессии из SQL Server;
- автоматический перенос состава корзины в форму контактов;
- адаптивная версия без изменения исходной стилистики.

## Требования

- Node.js 20 или новее;
- Microsoft SQL Server 2022 Express, экземпляр `SQLEXPRESS`;
- ODBC Driver 17 for SQL Server.

## Запуск

```powershell
npm install
npm run db:init
npm start
```

Сайт и API будут доступны по адресу `http://127.0.0.1:3000/`.

Параметры подключения находятся в `.env.example`. Для локального запуска используется Windows Authentication. Файл `.env` не добавляется в Git.

## Проверка

```powershell
npm test
```

Интеграционный тест создаёт отдельную временную SQL-базу, проверяет регистрацию, уникальность логина/email, сессию, выход и повторный вход, затем удаляет тестовую базу.
